My Work

Weather Dashboard

This weather app lets users search for city-specific weather data by entering a city name into the search field, and then fetching information from OpenWeatherMap API. It displays the current weather info like temperature, wind speed, and humidity, along with a 5-day forecast. The app saves previously searched cities in local storage and provides them as buttons for quick access.

BrewFinder

BrewFinder helps users locate breweries in a city, fetching brewery data via the Open Brewery Database API and displaying markers on a map using the Google Maps API. Users can view and select breweries from the map or view suggested breweries and their details, and get directions from their current location. It makes it easy to plan brewery visits and discover local breweries.

JavaScript Quiz

This JavaScript quiz app uses an object array to store multiple-choice questions, generates HTML elements for the quiz, and tracks the user's score, which is affected by a timer and deducted points for incorrect answers. At the end, users can enter their initials to save their score on the high scores page.
